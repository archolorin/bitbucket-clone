MobaXterm plugins
=================
In order to install these plugins, just download them and put them in the
same directory than MobaXterm executable.  If you need to enhance MobaXterm
with extra tools and commands, you can also use the MobApt package manager:
type "MobApt" (or "apt-get") inside MobaXterm terminal.
--https://mobaxterm.mobatek.net/plugins.html
